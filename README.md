# Resumatch

AI-powered ATS Resume Analyzer that evaluates resumes against job descriptions and provides structured feedback for improvement.

---

## 🚀 Features
- AI-based ATS resume scoring system
- PDF resume parsing and text extraction
- Skill, experience, and keyword analysis
- Personalized improvement suggestions
- Real-time resume evaluation
- Clean and responsive UI

---

## 🛠️ Tech Stack
- React.js
- TypeScript
- Zustand (State Management)
- AI APIs
- PDF Processing

---

## 📌 Project Overview
Resumatch is an intelligent resume screening tool that simulates Applicant Tracking System (ATS) evaluation. It analyzes resumes, compares them with job descriptions, and generates a structured score along with actionable feedback to improve job readiness.

---

## ⚙️ How It Works
1. Upload your resume (PDF)
2. Enter job description
3. AI analyzes matching skills & content
4. Get ATS score + detailed feedback
